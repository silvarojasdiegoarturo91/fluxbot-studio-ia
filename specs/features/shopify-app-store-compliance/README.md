# Shopify App Store Compliance

This feature keeps the Shopify App Store best-practices matrix, test cases and manual evidence in one place.

## Included surfaces

- Install, OAuth, session-token and embedded shell behavior.
- Billing, GraphQL Admin API and app-pricing behavior.
- Theme app extension and storefront config parity.
- Privacy, retention, support and factual listing evidence.

## Excluded families

- Payments apps.
- Purchase-option apps.
- Payment facilitator apps.
- POS integrations.
- Lending or capital products.
- Agency/freelancer marketplace surfaces.
