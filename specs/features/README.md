# Feature Specifications (Frontend / Shared)

This directory stores requirements grouped by feature folder.

## Convention

- One folder per feature: `specs/features/<feature-slug>/`
- Mandatory file: `requirements.md`
- Optional files:
  - `decisions.md`
  - `contracts.md`
  - `test-cases.md`

## Current feature folders

- `onboarding/`
- `shop-lifecycle/`
