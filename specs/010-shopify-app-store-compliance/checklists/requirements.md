# Specification Quality Checklist: Shopify App Store Compliance Matrix

**Feature**: [spec.md](../spec.md)

## Checklist

- [ ] CHK-001 The spec identifies the actual Shopify app family and excludes non-applicable families.
- [ ] CHK-002 The spec has a P1 journey for install/auth/session-token behavior.
- [ ] CHK-003 The spec has a P1 journey for billing, GraphQL Admin and theme extension behavior.
- [ ] CHK-004 The spec has a P2 journey for privacy, support and manual evidence.
- [ ] CHK-005 The spec explicitly ties the work to OpenSpec and SpecKit traceability.
- [ ] CHK-006 The spec distinguishes automated tests from manual evidence.
- [ ] CHK-007 The spec includes measurable success criteria.
- [ ] CHK-008 The spec clearly states assumptions that limit scope.
