# testRigor Shopify Testing

## Purpose
Companion guide for Shopify testing breadth and automation.

Primary source:
https://testrigor.com/shopify-testing/

## Rules
- Cover functional, UI, performance, security, SEO, compliance, webhook, regression, and CI testing.
- Mock Shopify API calls when unit tests should stay isolated.
- Use Shopify CLI where required for Functions, webhooks, and store-linked workflows.
- Prefer stable, maintainable test flows that reflect merchant reality.
- Use current state and live store constraints when testing mutations.

## Scope
- test design
- E2E automation
- webhook validation
- CI integration
- regression coverage
