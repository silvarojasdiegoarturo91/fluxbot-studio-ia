# WeArePresta Shopify AI Toolkit

## Purpose
Companion guide for AI-assisted Shopify app building.

Primary source:
https://wearepresta.com/building-shopify-apps-with-ai-in-2026-the-ultimate-guide-to-the-shopify-ai-toolkit/

## Rules
- Prefer plugin-based setup when available for auto-updates.
- Use manual skills only when you need tighter control.
- Use MCP to keep docs and schemas current.
- Validate with live/current Shopify schema before coding against it.
- Treat store operations as live and plan mutations carefully.

## Scope
- app scaffolding
- AI assistant setup
- MCP usage
- code validation
- store operations
